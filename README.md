# C++ OOP Journey

This repository documents a structured and practical journey toward mastering C++ and Object-Oriented Programming (OOP), with a strong focus on problem-solving, clean code practices, and real-world application.

## Overview

The goal of this repository is to build a solid foundation in C++ and gradually transition into advanced software engineering concepts through consistent practice and implementation.

The content is organized to reflect a logical learning progression:
- Revisiting core C++ fundamentals
- Understanding and applying OOP principles
- Reinforcing knowledge through hands-on mini projects

## Structure

- **01-review**  
  Covers essential C++ fundamentals such as arrays, functions, and core logic building.

- **02-oop-basics**  
  Introduces the building blocks of Object-Oriented Programming including classes, objects, and constructors.

- **03-oop-pillars**  
  Focuses on the four main OOP principles:
  - Encapsulation  
  - Abstraction  
  - Inheritance  
  - Polymorphism  

- **04-mini-projects**  
  Contains practical implementations that combine multiple concepts into real use cases.

## Approach

This is not a collection of random code snippets.  
Each file is written with the intention of understanding how and why things work, not just how to write them.

The repository is updated incrementally to reflect continuous learning and improvement.

## Objective

The long-term objective is to reach a professional level in software engineering, with the ability to design, build, and maintain scalable and efficient systems using C++.
